# Desafio_gonzaguinnha 

## 📑 Relatório de Desenvolvimento: Site "O Que É, O Que É?"
Este projeto foi desenvolvido como um desafio acadêmico para criar um site funcional e estilizado com Bootstrap, utilizando como base a letra da música "O Que É, O Que É?", do compositor Gonzaguinha. O objetivo foi traduzir a mensagem da obra para uma interface web moderna e organizada.

### 🛠️ O que foi desenvolvido no site
Estrutura de Navegação: Implementei uma navbar fixa com o nome do projeto, permitindo o acesso rápido às seções de filosofia e contato.

Seção de Destaque (Carrossel): Configurei um carrossel de imagens para ilustrar visualmente os temas da música (alegria, luta e o coração). Ajustei o tamanho das imagens para 300px para garantir que o conteúdo principal ficasse visível sem a necessidade de rolar muito a página.

Conteúdo Principal e Cards: Dividi o corpo do site em colunas. Na principal, apresento uma interpretação da letra e um botão interativo para o usuário dar sua própria definição de "vida". Usei cards para destacar conceitos como "O Aprendiz" e "A Luta".

Interatividade e Sidebar: Incluí uma barra lateral com formulário de newsletter e uma lista das "Faces da Vida" citadas na obra, além de um modal (janela flutuante) para o formulário de contato.

Legibilidade: Configurei todo o corpo do site para que as fontes sejam obrigatoriamente pretas, garantindo 100% de contraste e facilidade de leitura sobre os fundos claros.

## 🎨 A Importância da Paleta de Cores
A parte visual foi o maior desafio do projeto. Para chegar ao resultado esperado, utilizei a ajuda do Gemini de forma estratégica:

O suporte da IA foi essencial para a transição da paleta de cores. Eu selecionei e enviei imagens de paletas específicas que eu desejava utilizar. A partir dessas imagens, o Gemini extraiu os códigos hexadecimais exatos (Burgundy, Lapis Lazuli, Cherry Blossom) e os aplicou diretamente no CSS. Essa ajuda foi fundamental para que eu pudesse sair do padrão comum e criar uma identidade visual única, que mistura a sobriedade do azul e vinho com a vivacidade do rosa, refletindo exatamente o espírito da música.
